# Images Folder

This folder contains the slideshow images for the Mann Ki Baat section.

## Required Images:
- slide1.jpg - Latest Mann Ki Baat episode
- slide2.jpg - Previous episode 
- slide3.jpg - Previous episode
- slide4.jpg - Previous episode
- slide5.jpg - Previous episode

## Image Specifications:
- Recommended size: 1200x600 pixels (2:1 aspect ratio)
- Format: JPG or PNG
- File size: Keep under 500KB for optimal loading

## Usage:
Replace the placeholder images with your actual Mann Ki Baat images. The slideshow will automatically cycle through all 5 images.

The images should be high-quality photos related to Mann Ki Baat episodes, featuring the Prime Minister or relevant government content.
